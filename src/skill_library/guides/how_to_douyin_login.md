# 如何实现抖音验证码登录

## 流程

1. 打开 `https://www.douyin.com/discover`
2. 点击页面右上角红色“登录”按钮，打开登录弹窗
3. 自动填写用户提供的手机号
4. 自动点击验证码输入框右侧的获取验证码
5. 用户在浏览器中手动输入短信验证码

## 使用示例

```python
from skill_library.others.douyin_login import run

run("13574133406")
```

自然语言示例：

```text
抖音登录，电话号码是13574133406
```

## 注意

该技能不会勾选协议，也不会自动输入短信验证码或在最终登录完成前保存 cookie。
